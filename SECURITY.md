# Security Policy

We take security seriously. If you discover a vulnerability in **AlpURL**, please let us know so that we can resolve it promptly.

---

## Supported Versions

Only the latest active release is supported for security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

---

## Reporting a Vulnerability

Please do not report security vulnerabilities via public GitHub issues. Instead, report them privately using the following process:

1. Send an email to **rapidrevisionhub@gmail.com** containing details of the vulnerability.
2. Include a descriptive summary, details on the impact, steps to reproduce, and any proof-of-concept scripts.
3. We will respond within 48 hours to confirm receipt and coordinate a patch timeline.
4. Once resolved, we will publish a security advisory and credit you for the discovery.
